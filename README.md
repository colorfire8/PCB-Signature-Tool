# PCB-Signature-Tool

用于在嘉立创EDA / EasyEDA 的 **PCB 丝印层**生成并插入作者签名信息图，适合在作品板子上留下署名与仓库信息。

仓库：`https://github.com/colorfire8/PCB-Signature-Tool`

## 功能

- **生成签名信息图**：作者（可用手写签名替换）、版本号、仓库二维码、仓库文本、开源协议、致谢信息
- **多仓库链接 & 多二维码**：支持配置 GitHub / Gitee / 嘉立创项目仓库链接，可单一或多个二维码同时显示（默认仅 GitHub）
- **可拖拽排版预览**：在预览画布内可拖动“二维码块 / 文字块”微调布局，插入时沿用该布局
- **插入丝印**：
  - 彩色：以图片对象方式插入 `TOP_SILKSCREEN`
  - 非彩色：矢量化后插入丝印（支持反色）
- **预览与配置保存**：在弹窗内预览效果，并自动保存上次填写的信息（下次打开自动恢复）

## 使用方法
![test](images/test.png)
1. 导入插件后，在嘉立创EDA 的 PCB 编辑器中点击：`高级 → PCB签名工具 → PCB签名工具`
2. 填写仓库链接（GitHub 必填或填其它之一）、作者、版本号（可选手动填写协议）
3. 选择二维码来源（可多选）、颜色模式（默认/反色/彩色）与大小，查看预览
4. （可选）添加手写签名：在线绘制或上传图片，预览中会替换作者那一行
5. （可选）在预览画布中拖动“二维码块 / 文字块”微调布局
6. 点击“生成并插入作者签名”，按提示在 PCB 画布中单击放置
